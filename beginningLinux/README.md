# Linuxことはじめ
## Linuxって何ですか
- WindowsとかMacとかのOSです。
  - ただ、いくつか流派があったりします。変なたとえですが、関西弁は土佐弁や薩摩弁と同様に方言のひとつですが、細かくは京都弁と大阪弁と神戸のやつとあります、みたいな（なんか違う）
  - ↑ いや別に英語は日本語や中国語みたいに言語だけど、アメリカ英語とイギリス英語とオージー英語とインド英語とありますみたいな感じでもいいです。（本質じゃないし）
  - えっと、実際の流派とすると、UbuntuとかCentOSとかRedHatとかあります。（ディストリビューションという。そしていくつかのディストリビューションがDebian系、RedHat系とかまとめられたりしてます）
  - ここは本質じゃないから最初は気にしなくていい。。。
- 特徴はマルチユーザー・マルチタスク。普通はパソコンを（まさにPersonalなComputerとして）ひとりで使いますが、共用マシンでガシガシ計算などをするためにみんなで接続して同時にタスクを実行するのに使われます。
- リモートで使うので、マウスで選択するようなグラフィカルなインターフェース（Graphilal User Interface＝GUI）でなく、コマンドと呼ばれる命令（通称：呪文）を書いて何かを実行するのがメインです。
  - GUIもあります。
  - 実際のところ、GUIで使うコンピューターリソースを計算するのに使うのでGUI要らないみたいなのが極論として（or 半ばジョークとして）言われたりします
  - → なのでどういう呪文を唱えると何ができるか、に慣れましょう、というのがこのコンテンツの目的です。。。
## とりあえずスタートしてみる
- まずはターミナルと呼ばれるソフトを使ってLinuxマシンにログインしないといけないのですが、とりあえずそこはクリアした、ということにしておく。。。
- 実際のところは、WindowsだとTera Termとか、Macだとアクセサリの下のターミナル というソフトを立ち上げてログインしたりするのですが。。。

### 最初の呪文を唱える
- ログインした場所でどんなファイルがあるかを見るコマンドが`ls`です。
```
ls
```
- まぁえてして何も出なかったりするんですが。
